¿Que es HTML?

HTML es el lenguaje con el que se define el contenido de las páginas web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript).
